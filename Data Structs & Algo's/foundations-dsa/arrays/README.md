# Arrays
Arrays are the #1 foundation for DSA. In Python, arrays are usually represented as **lists**.

## Goals
- Become automatic with indexing, traversal, and in-place updates
- Recognize core patterns: two pointers, sliding window, prefix sums
- Solve common interview-style array problems confidently

## Folder Guide (Patterns)
- **array-basics/** → indexing, traversal patterns, in-place updates
- **common-array-methods/** → Python list operations (append/pop/slicing/sort)
- **frequency-counting/** → using dict/set with arrays
- **prefix-sums/** → fast subarray/range calculations
- **sliding-window/** → best for subarray/substring length problems
- **two-pointers/** → left/right pointer movement problems
- **array-sorting/** → sorting tricks and sort-based solutions
- **intervals/** → merge/insert interval patterns
- **matrix-2d-arrays/** → 2D traversal + grid-style problems
- **array-stack/** → monotonic stack patterns using lists
- **array-other/** → classic “must know” problems

## Arrays = Lists (Python quick notes)
- Indexing: `arr[i]`
- Append: `arr.append(x)`
- Pop: `arr.pop()`
- Insert: `arr.insert(i, x)`
- Remove: `arr.remove(x)`
- Sort: `arr.sort()` (in-place) vs `sorted(arr)` (new list)
